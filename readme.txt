= Description ==

With this plugin you Pay with Paystack on Paid Membership Pro Plugin.


= Plugin Features =

*   __Accept payment__ via  Debit card, Bank account, USSD, MPESA.
*    __Seamless integration__ with Paid Memebership Pro.
*    __Different currencies__ USD, NGN, KES, GHC.

= Manual Installation =
1.  Download the plugin zip file
2.  Login to your WordPress Admin. Click on "Plugins > Add New" from the left hand menu.
3.  Click on the "Upload" option, then click "Choose File" to select the zip file from your computer. Once selected, press "OK" and press the "Install Now" button.
4.  Activate the plugin.
5.  Under Paid Membership Pro payments, configure your api keys under api keys settings.
